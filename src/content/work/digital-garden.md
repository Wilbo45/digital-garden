## Overview

This site itself is the first entry in the Work section. It's built with Astro, styled with a custom VPDS-inspired design system, and deployed to GitHub Pages.

## Goals

- A permanent home for long-form writing
- A platform for publishing interactive data visualisations
- A design system I can build on iteratively

## Technical decisions

**Astro** was chosen for its content-first architecture, zero-JS-by-default approach, and flexibility to embed interactive components (React, Svelte, D3) only where needed.

The design system is inspired by the Visa Product Design System — deep navy palette, Inter typography, a tight spacing scale, and minimal ornamentation.
